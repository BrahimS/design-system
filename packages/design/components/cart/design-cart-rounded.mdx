The cart can have rounded corners.
There are three options for this property :

| rounded   | example                                                                               | description                       |
|-----------|---------------------------------------------------------------------------------------|-----------------------------------|
| `all`     | ![Preview of cart with sm size](design-images/cart/cart-rounded/cart-rounded-all.png) | All corners will be rounded       |
| `none`    | ![Preview of cart with sm size](design-images/cart/cart-rounded/cart-rounded-none.png)| No rounded corners                |
| `top`     | ![Preview of cart with sm size](design-images/cart/cart-rounded/cart-rounded-top.png) | Only top corners will be rounded  |

> preview with the ODS theme `blue-jean`
