## Preview
It is a smart component without UI.

## Description
The checkbox component is used to check one item. It can be used to turn an option on or off.
This component is a smart component that need a child as `tile` component, or any other component
that can be `checked`.

## Zoning
There is no UI for that component. Child component has the UI.

## Colors
Colors don't apply directly to Checkbox but come from child component (ie.toggle, tile and checkbox-button).
