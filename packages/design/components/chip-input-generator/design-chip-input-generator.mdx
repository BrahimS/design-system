## Description
The chip-input-generator is a complex form part that generates some chips to display keywords, tags, etc.

## Zoning
It contains three parts, an [Input]() followed by a [Button](), then a list of [Chips]() are displayed with the action 
of clicking on the Button after filling the Input with text.
