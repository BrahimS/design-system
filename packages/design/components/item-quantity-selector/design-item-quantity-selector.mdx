## Description
The Item Quantity Selector is a component to be used for selecting an item and a quantity. It has different UIs, but 
they all have the same purpose.

This component uses the [Quantity Selector]() and [Selectable tile]() components.

## Zoning
The component is a clickable tile with description of the pointed item, and a quantity container attached to it.
