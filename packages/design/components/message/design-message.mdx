## Description
The Message component displays a message in a coloured container. This is only a read-only component, with an option to 
remove the message itself.

## Zoning
The component has a main content in it, with an optional cross icon in the top right corner of it.

## Colors
All available colors can be applied on the message : default, primary, text, accent, error, warning, success, info and promotion
