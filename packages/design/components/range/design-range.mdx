## Description
The Range component is a specific input type that allows users to select a value (or a couple of values) from a range of values.
A range can be `disabled` if you don't want to allow user to select a value.
The range `value` can be set at start as a `numeric` or a `couple of numerics`.
The component can be defined with `min`, `max` and `step` attributes.
You can display the range in `flex` mode.

## Zoning
The component has 2 optional contents: a start and a end.

<osds-range min="0" max="10" step="1" value="5" style={{ pointerEvents: 'none' }}>
  <span slot='start-bound'>start</span>
  <span slot='end-bound'>end</span>
</osds-range>

## Colors
All available colors can be applied on the range : default, primary, text, accent, error, warning, success, info and promotion

## Sizes
It has for now only one size `md`.
