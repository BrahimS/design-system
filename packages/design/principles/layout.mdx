## Grid
_Every layout should sit on a grid that brings order and hierarchy to the interface. Define a grid separately for mobile, tablet and desktop devices with columns, gutters, and margins so your interface can adapt to any platform easily._

## Breakpoints

| name             | purpose                                                           | size(rem) | size(px)   |
|-- -------------- | ----------------------------------------------------------------- | --------  | ---------- |
| `--ods-mq-sm`    | For small viewports as mobile devices (in portrait)               | 36rem     | 576px      |
| `--ods-mq-md`    | For smartphones (in landscape) or tablet (in portrait) viewports  | 48rem     | 768px      |
| `--ods-mq-lg`    | For small desktops or tablets (in landscape) viewports            | 62rem     | 992px      |
| `--ods-mq-xl`    | For desktops viewports                                            | 75rem     | 1200px     |
| `--ods-mq-xxl`   | For widescreen desktops
