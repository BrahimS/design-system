
# Theming
_All components uses the theming system_

the theming is composed of different principles made of theses concepts :
- color palettes
- color
- contrast
- sizes

Defining a theme means set a color palettes for each color :

| color    | example |
|---------|-------------|
| default | use a gray palette             |
| primary | blue palette |
| accent  | green palette |
| error   | red palette |
| warning | orange palette |
| info    | blue palette |
| success | green palette |


## Color of components

Each component uses the theming system.
A color like `primary` for the component means that the color palette defined inside the theme will be used.
