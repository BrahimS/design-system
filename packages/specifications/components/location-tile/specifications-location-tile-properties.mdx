| Name           | Type                                    | default      | Description                                                                     |
|-- ------------ | -------------------------------------   | ------------ | ------------------------------------------------------------------------------- |
| `[city]`       | `string`                                | -            | city to display on the tile                                                     |
| `[code]`       | `string`                                | -            | code of the location (like a datacenter code)                                   |
| `[color]`      | **_OdsThemeColor_**                     | `default`    | main color<br />_See design components principles_                              |
| `[disabled]`   | `boolean`                               | `false`      | disabled state<br />_See design components principles_                          |
| `[flex]`       | `boolean`                               | `false`      | full width or not                                                               |
| `[iso]`        | `ODS_COUNTRY_ISO_CODE` from `@ovhcloud/ods-core` | -            | country (+ country flag) to display on the tile                                 |
| `[checked]`    | `boolean`                               | `false`      | displayed as selected with a different color. No clickable feature management   |
| `[checking]`   | `boolean`                               | `false`      | used to display the tile while being checked                                    |
| `[size]`       | **_OdsLocationTileSize_**               | `md`         | size<br />_See design components principles_                                    |
| `[variant]`    | **_OdsLocationTileVariant_**            | `vertical`   | Location tile variant                                                           |
