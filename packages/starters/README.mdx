## ODS Starters

You built some application starters ready to use with the most used framework.
These samples are made in order to get started more efficiently.
They integrate `ODS` components and show how to use them.

- [@ovhcloud/ods-starter-react](/story/code-starters-react--page): classic `React` application
- [@ovhcloud/ods-starter-react-vite](/story/code-starters-react-vite--page): `React` application with `Vite` compiler
- [@ovhcloud/ods-starter-vue-vite](/story/code-starters-vue-vite--page): `VueJs` application with `Vite` compiler
